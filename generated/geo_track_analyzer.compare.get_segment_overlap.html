geo_track_analyzer.compare.get_segment_overlap

geo_track_analyzer.compare.get_segment_overlap(base_segment: GPXTrackSegment, match_segment: GPXTrackSegment, grid_width: float, max_queue_normalize: int = 5, allow_points_outside_bounds: int = 5, overlap_threshold: float = 0.75) → list[SegmentOverlap]

Compare the tracks of two segements and caclulate the overlap.

Parameters:

  • base_segment – Base segement in which the match segment should be found

  • match_segment – Other segmeent that should be found in the base segement

  • grid_width – Width (in meters) of the grid that will be filled to estimate the overalp.

  • max_queue_normalize – Minimum number of successive points in the segment between two points falling into same plate bin.

  • allow_points_outside_bounds – Number of points between sub segments allowed for merging the segments.

  • overlap_threshold – Minimum overlap required to return the overlap data.

Returns:

list of SegmentOverlap objects.
