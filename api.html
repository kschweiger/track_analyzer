API Reference

Tracks

geo_track_analyzer.GPXFileTrack(gpx_file[, ...])

Track that should be initialized by loading a .gpx file

geo_track_analyzer.FITTrack(source[, ...])

Track that should be initialized by loading a .fit file

geo_track_analyzer.PyTrack(points, ...[, ...])

Track that should be initialized from python objects

geo_track_analyzer.ByteTrack(bytefile[, ...])

Track that should be initialized from a byte stream

geo_track_analyzer.SegmentTrack(segment[, ...])

Track that should be initialized by loading a PGXTrackSegment object

Model

geo_track_analyzer.model.Position2D(...)

Position in a 2D latitude / longitude space

geo_track_analyzer.model.Position3D(...)

Position in a 3D latitude / longitude / elevation space

geo_track_analyzer.model.ElevationMetrics(...)

Collection of elevation related metrics

geo_track_analyzer.model.SegmentOverview(...)

Collection of metrics for a Segment

geo_track_analyzer.model.SegmentOverlap(...)

Represent the overlap between two segments

geo_track_analyzer.model.PointDistance(...)

Represents a distance calculation to a point in a GPX track.

Enhancer

geo_track_analyzer.OpenTopoElevationEnhancer([...])

Use the/a OpenTopoData API (https://opentopodata.org) to enhance a GPX track with elevation information.

geo_track_analyzer.OpenElevationEnhancer([url])

Use the/a OpenElevation API (https://open-elevation.com) to enhance a GPX track with elevation information.

geo_track_analyzer.get_enhancer(name)

Get a Enhance object for a specific enpoint by passing a distinct name

Visualizations

geo_track_analyzer.visualize.plot_track_3d(data)

geo_track_analyzer.visualize.plot_track_line_on_map(data, *)

Plot the track line on a map.

geo_track_analyzer.visualize.plot_track_enriched_on_map(data, *)

Plot the track line enriched with additional information (as z-axis) on a map.

geo_track_analyzer.visualize.plot_track_2d(data, *)

Elevation profile of the track.

geo_track_analyzer.visualize.plot_track_with_slope(data, *)

Elevation profile with slopes between points.

geo_track_analyzer.visualize.plot_segments_on_map(data, *)

Plot track line on map.

Track comparisons

geo_track_analyzer.compare.check_segment_bound_overlap(...)

Check if segments are within the bounds of a reference segment

geo_track_analyzer.compare.get_segment_overlap(...)

Compare the tracks of two segements and caclulate the overlap.

geo_track_analyzer.compare.convert_segment_to_plate(...)

Takes a GPXSegement and fills bins of a 2D array (called plate) with the passed bin width.
